# AI-Powered Smart Public Service CRM (PS-CRM)

A modern civic-tech platform for grievance management.

## Tech Stack
- **Frontend:** React, Tailwind CSS, Vite, Lucide React, Recharts, Framer Motion
- **Backend:** Node.js, Express, MongoDB
- **Auth:** JWT and Role-Based Access Control

## Getting Started

### Backend Setup
1. `cd server`
2. `npm install`
3. Create `.env` file (template provided)
4. `npm start` (or `node server.js`)

### Frontend Setup
1. `cd client`
2. `npm install`
3. `npm run dev`

## Roles
- **Citizen:** Report issues and track progress.
- **Admin:** Oversight, analytics, and department assignment.
- **Officer:** Resolution and progress updates.
